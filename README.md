# AI Email Transaction Parser

This project is an AI-powered system that extracts and organizes financial transaction data from email notifications.

It is designed to take unstructured transaction emails, identify key details such as merchant, amount, and date, and convert that information into structured data that can be tracked or analyzed.

## Features
- Extracts transaction details from email alerts
- Organizes unstructured text into usable data
- Supports automation workflows
- Built to reduce manual financial tracking

## Tech Used
- OpenAI API
- Gmail / email filters
- Make.com
- Excel or spreadsheet logging

## Project Goal
The goal of this project is to show how AI can be used to automate everyday financial workflows and turn simple notifications into actionable data.
